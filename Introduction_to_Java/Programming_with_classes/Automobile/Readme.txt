Задача 2 раздела агрегация и композиция
Условие:
Создать объект класса Автомобиль, используя классы Колесо, Двигатель. Методы: ехать, заправляться, менять колесо, вывести на консоль марку автомобиля.
Решение:
Automobile.java - класс представляющий автомобиль
Wheel.java - класс представляющий колесо
Engine.java - класс представляющий двигатель
Demo.java - класс демонстрирующий возможности класса Automobile.
