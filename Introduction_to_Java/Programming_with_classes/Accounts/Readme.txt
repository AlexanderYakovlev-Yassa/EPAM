Задача 4 раздела агрегация и композиция
Условие:
Счета. Клиент может иметь несколько счетов в банке. Учитывать возможность блокировки/разблокировки счета. Реализовать поиск и сортировку счетов. Вычисление общей суммы по счетам. Вычисление суммы по всем счетам, имеющим положительный и отрицательный балансы отдельно.
Решение:
Automobile.java - класс представляющий автомобиль
Wheel.java - класс представляющий колесо
Engine.java - класс представляющий двигатель
Demo.java - класс демонстрирующий возможности класса Automobile.

Account.java - класс представляющий счет клиента в банке
Accounts.java - класс агрегирующий счета клиентов
AccountStatus.java - класс перечесление статусы счетов
AccountTypes.java - класс перечесление типы счетов 
Client.java - класс представляющий клиента
Demo - класс демонстрирующий возможности класса Accounts
