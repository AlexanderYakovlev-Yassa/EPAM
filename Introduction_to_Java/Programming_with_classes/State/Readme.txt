Задача 3 раздела агрегация и композиция
Условие:
Создать объект класса Государство, используя классы Область, Район, Город. Методы: вывести на консоль столицу, количество областей, площадь, областные центры.
Решение:
City - класс представляющий город
District - класс представляющий район
Region - класс представляющий область
State - класс представляющий государство
Demo.java - класс демонстрирующий возможности класса State.
