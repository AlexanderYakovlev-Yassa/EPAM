Задача 5 раздела агрегация и композиция
Условие:
Туристические путевки. Сформировать набор предложений клиенту по выбору туристической путевки различного типа (отдых, экскурсии, лечение, шопинг, круиз и т. д.) для оптимального выбора. Учитывать возможность выбора транспорта, питания и числа дней. Реализовать выбор и сортировку путевок.
Решение:
Base.java - класс агрегирующий объекты типа Tour, путевки
Demo.java - класс демонстрирующий возможности класса Base
Food.java - класс перечесление типы питания
Tour.java - класс представляющий возможные путевки
Transportation.java - класс перечесление типы транспорта
Type.java - класс перечесление типы путевок
