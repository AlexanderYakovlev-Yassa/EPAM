Задача 1 темы агрегация и композиция
Условие:
Создать объект класса Текст, используя классы Предложение, Слово. Методы: дополнить текст, вывести на консоль текст, заголовок текста.
Решение:
Word.java - класс представляющий слово
Sentence.java - класс представляющий предложение
Text.java - класс представляющий текст
Demo.java - класс демонстрирующий возможности классов Word, Sentence, Text.
